# Contributing to ConVol

Thanks for your interest in contributing! ConVol is built by the fan 
convention community, for the fan convention community.

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a branch for your feature or fix
4. Make your changes
5. Submit a pull request

## Branch Naming

- `feature/your-feature-name`
- `fix/what-youre-fixing`
- `docs/what-youre-documenting`

## Pull Requests

- Keep PRs focused — one feature or fix per PR
- Write a clear description of what your PR does and why
- Reference any related issues

## Reporting Issues

Use GitHub Issues to report bugs or suggest features. Please include:
- What you expected to happen
- What actually happened
- Steps to reproduce

## Code of Conduct

Be kind. This is a community project built by convention volunteers 
for convention volunteers. Treat everyone with the same respect you'd 
want on the con floor.

## Questions?

Open a GitHub Discussion or file an issue tagged `question`.
